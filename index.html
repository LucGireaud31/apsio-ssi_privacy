Rêgles de confidentialité d'Apsio SSI

1. Données utilisateur

Le but d'Apsio SSI est de regrouper toutes les données personnelles d'un utilisateur et de les partager à des tiers de la façon la plus simple et rapide possible.

Voici la liste exaustive des informations stockées dans l'application
  • Nom
  • Prénom
  • Date de naissance
  • Adresse mail
  • Adresse postale
  • Numéro de téléphone
  • Sexe,
  • Carte(s) bleue(s)
Il est à noté qu'en aucun cas ces informations sont requises au fonctionnement de l'application, il est juste recommandé de stocker le plus de données possible pour que son utilisation soit optimale.

Les données inscrites par l'utilisateur sont stockés seulement en local dans le téléphone et ne sont jamais transmises a moins que vous ne les partagiez.

Pour rajouter une couche de sécurité ces données sont cryptées en utilisation le chiffrement AES et un code pin modifiable par l'utilisateur qui devra l'inscrire à chaque ouverture d'Apsio SSI pour décrypter les données.

2. Autorisations

- Caméra : Une fonctionnalité permet de partager ses données en scannant un QR Code généré par l'établissement compatible SSI. Il est donc recommandé d'autoriser l'application à accéder à la caméra du téléphone. Sans cela l'utilisateur pourra quand même partager ses données en générant un QR Code ou en copiant les données dans le presse papier

- Stockge du téléphone : Il est possible de choisir une photo de profil, elle n'est pas partageable et est cryptée comme les autres données. Elle n'apporte aucune fonctionnalité en plus.

3. Connexion

Il faut enregistrer un code de 4 chiffres pour pouvoir se connecter à l'application.
Ce code n'est jamais divulgé et est enregistré sur l'appareil en lui passant un algorithme de hashage (sha256).
